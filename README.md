# Supermarket-Manager
This app main purpose is to facilitate supermarket operations.
It was coded as part of TC1031.1 class.

## Key Features 🗝 
* Based on Double Linked Lists and .txt handling.
* Bubble Sort Algorithm Implementation.
* Structured paradigm.
